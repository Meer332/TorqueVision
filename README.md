# TorqueVision

TorqueVision is a realistic automotive dashboard simulator built using HTML, CSS, JavaScript and SVG.

## Features

- SVG Analog Gauges
- Multi-Car Support
- Engine Sound System
- Drive Modes
- Gearbox Simulation
- Shift Lights
- Boot Animation
- Temperature, Fuel, Oil and Voltage Gauges

## Technologies

- HTML5
- CSS3
- JavaScript
- SVG
